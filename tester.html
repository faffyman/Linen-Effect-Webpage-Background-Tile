Blue Weave/Linen Effect

Creating a linen effect backdrop image - similar to that default mac wallpaper is not that difficult.

Simply follow these steps - make your own colour choices and tweak the values along the way to suit yourself

  • Start with a large blank square, I use 256x256px
  • Call your first layer - vertical and Fill with the colour of your choice - eg. #8a8a8a
  • We need to Add Noise. Filter > Noise > Add Noise. Select monochromatic/guassian mid 60% range again
  • Apply a motion blur to this layer 90 degrees and distance of about 50-100 (I used 60)
  • Create a new layer (horizontal)and fill with a contrasting color, e.g. #cfcfcf
  • Again add noise - select a different percentage value for this layer
  • Apply a motion blur again. This time 0 degrees and a slightly different amount.
  • Set Layer blending mode to "screen" on the top most layer
  • You can add a differnt base colour layer on the bottom - and make both the horizontal and vertical layers blending modes "screen" to see hints of this colour

Now the harder bit - making it tile

  • Use the Filter >> Offset.
    Set the horizontal and vertical offsets to half your image width and height respictively
  • The undefined areas should be set to wrap around
  • Now withe the filter applied, use the clone tool to remove the obvious join lines
  • Perform the offset again - hopefully there will be no join lines, repeat thi sprocess until you are happy with the results
  • You final image will most likely have a border area which looks "wrong" Just crop your final image inside this
